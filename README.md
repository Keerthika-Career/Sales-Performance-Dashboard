📊 Sales Performance Dashboard (Power BI)

🔍 Overview
This project presents an interactive Sales Performance Dashboard built using Power BI to analyze sales performance across employees, regions, and time. The dashboard enables quick decision-making by providing clear visual insights into key business metrics.

🎯 Objective
To monitor and evaluate sales performance, identify top-performing employees, and analyze regional trends to support data-driven business decisions.

📌 Key Features
•	📊 Sales by Employee Analysis
•	🌍 Sales by Region Distribution
•	📈 Monthly Sales Trend Analysis
•	📦 Deal Size Analysis
•	🔄 Sales Value vs Sales Volume Comparison
•	🎛 Interactive Filters (Employee, Region, Year)

🛠 Tools & Technologies Used
•	Power BI (Data Visualization)
•	Excel (Data Source)
•	DAX (Calculated Measures)

📊 Key Insights
•	East region contributes the highest share of total sales (~60%)
•	David is the top-performing employee
•	Sales peak observed in February, indicating seasonal trends
•	Smaller deal sizes contribute to higher transaction volume

🚀 Conclusion
This dashboard provides a comprehensive view of sales performance, helping stakeholders identify trends, track KPIs, and improve strategic decision-making.
